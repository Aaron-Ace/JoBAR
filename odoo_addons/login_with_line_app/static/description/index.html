Login With Line App

This module Helps You To Login With your Account in Line Chat App.


Configuration.

In General Settings under Integrations tab, enable OAuth Authentication and select OAuth Authentication page.

Line App Integration.

Inside the Providers setting page, you can see a new option as Line Chat.
By clicking the option you can add your line app details in next page.

Adding Line App Details

You need to add your client id in the field to integrate you line app.
To create a client id you need to Create your LINE Login channel and you need to use the Channel ID you got from there as the client id.
and make sure the the ALLOWED option is enabled.

After Adding details and enabling allowed button, you can see a new button "Log in with Line" in signin page.
By clicking the button you will be redirected to the line login page.

There you can login with your Line chat app credentials.
